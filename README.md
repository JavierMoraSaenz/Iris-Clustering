# Iris-Clustering
#  Clustering con K-Means y PCA

Proyecto práctico de Machine Learning no supervisado utilizando K-Means Clustering y PCA para identificar grupos y reducir dimensionalidad en el dataset Iris.

## 📌 Objetivo

Aplicar técnicas de clustering para identificar patrones y agrupaciones dentro de los datos, así como utilizar PCA para simplificar y visualizar información multidimensional.

## 🛠 Tecnologías utilizadas

- Python
- Pandas
- Matplotlib
- Seaborn
- Scikit-learn
- Jupyter Notebook

## 📊 Temas desarrollados

### 🔹 K-Means Clustering
- Entrenamiento del modelo
- Agrupación de datos
- Predicción de clusters
- Visualización de centroides

### 🔹 Método del Codo
- Cálculo de WCSS
- Selección del número óptimo de clusters

### 🔹 Silhouette Score
- Evaluación de calidad de agrupamiento
- Comparación de diferentes valores de K

### 🔹 PCA (Principal Component Analysis)
- Reducción de dimensionalidad
- Transformación de variables
- Visualización en 2 dimensiones

## 📈 Resultados

Se identificaron agrupaciones naturales dentro del dataset Iris y se evaluó la calidad de los clusters utilizando métricas de distancia y cohesión.

## 🚀 Aprendizajes

- Implementación de K-Means con Scikit-learn
- Evaluación de clusters
- Uso de PCA para simplificar datos
- Visualización de modelos no supervisados
- Interpretación de agrupaciones
